Scarpe & Co - Web Application ASP.NET Core MVC

Descrizione del progetto:
Questo progetto consiste in una Web Application sviluppata in ASP.NET Core MVC per la gestione e la vendita di prodotti di un negozio di scarpe da tennis. 
L'applicazione permette di archiviare e visualizzare informazioni relative agli articoli, tra cui nome, prezzo, descrizione e immagini, offrendo una vetrina semplice e funzionale per l'utente finale.

-Gestione prodotti: nome, prezzo, descrizione dettagliata, immagine di copertina e due immagini aggiuntive.
-Vetrina prodotti: visualizzazione della lista dei prodotti con immagine di copertina, nome e prezzo.
-Pagina dettaglio prodotto: visualizzazione di tutte le informazioni relative al singolo prodotto.

Tecnologie utilizzate:
-Backend: ASP.NET Core MVC
-Frontend: HTML, CSS, Razor Views
-Pattern: Model-View-Controller (MVC)
